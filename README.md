## IAS Cipher Website (PHP)

Simple PHP website for our group project:

- Main menu
- Pick a cipher
- Encrypt / Decrypt

### Structure

- `index.php`: menu (links to each cipher)
- `common/`: shared layout + helpers
- `ciphers/`: one folder per cipher (each member works in their own folder)

### How to run (XAMPP)

1. Put this folder inside `htdocs`
2. Start Apache in XAMPP
3. Open `http://localhost/IAS(HILL)/`

### Team workflow (GitHub)

- Each person edits only their cipher folder inside `ciphers/`
- Keep shared files in `common/` stable
